Basics

:tag: | :element:

HTML describes the structure of web pages. The HTML code that are within angle brackets (< and >) are called HTML elements. Elements are usually made up of two tags: an opening tag and a closing tag.

The terms “tag” and “elements” are often used interchangeably. Strictly speaking, however, an element comprises the opening tag and the closing tag and any content that lies between them.

For example:

<p> This is a paragraph. </p>
This whole thing can be considered an element. while only

and

can be considered tags.

:attribute:

Attributes tell us more about elements. They provide additional information about the contents of an element. They are made up of two parts: * a name * a value

For example:

<p lang="en-us"> Paragraph in English</p>

The attribute name indicates what kind of extra information you are supplying about the element’s content. It is written in lowercase. The value is the information or setting for the attribute. It should be placed in double quotes.

Most attribute values are either pre-defined or follow a stipulated format.
